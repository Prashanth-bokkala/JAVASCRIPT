Game Rules

Welcome to "THE 50 GAME"! The rules of the game are simple:

  1. Players take turns to click the "Player A" or "Player B" buttons to score runs.
  2. A random number of runs (between 1 and 6) is added to the current player's score on each turn.
  3. The game ends when one of the players reaches or exceeds the target score of 50.
  4. The player with the highest score at the end of the game wins.
  5. The game history table displays the results of the last three games played.

Good luck and enjoy playing "THE 50 GAME"!
